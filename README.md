# SoundBoard

SoundBoard is an easy to use soundboard whit a modern design. it also makes it easy to add yourn own sound

## Change sound

Add your own sounds by changing the source. Make sure your sound is in the right folder!

```javascript
<audio id="a">
  <source src="sounds/jeff.mp3" type="audio/mpeg">
</audio>
```

## Add more sounds

Add more sounds by simply copying this code and paste in your index. this gives you another row and the option to ad 3 more sounds.

```html
<div class="row">
      <div class="col-lg-4">
        <img class="bd-placeholder-img rounded-circle" src="images/jeff.jfif" width="140px" height="140px"> 
        <h2>My name is Jeff</h2>
        <button class="btn btn-secondary" onclick="sound1()" type="button">play ▶️</button>
      </div><!-- /.col-lg-4 -->
      <div class="col-lg-4">
        <img class="bd-placeholder-img rounded-circle" src="images/jojo.jpg" width="140px" height="140px"> 
        <h2>JoJo</h2>
        <button class="btn btn-secondary" onclick="sound2()" type="button">play ▶️</button>
      </div><!-- /.col-lg-4 -->
      <div class="col-lg-4">
        <img class="bd-placeholder-img rounded-circle" src="images/air-horn.jpg" width="140px" height="140px"> 
        <h2>Air-Horn</h2>
        <button class="btn btn-secondary" onclick="sound3()" type="button">play ▶️</button>
      </div><!-- /.col-lg-4 -->
    </div><!-- /.row -->
```
